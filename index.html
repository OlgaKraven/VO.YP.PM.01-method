1. Теория

В этом документе собраны краткие методические рекомендации для начального курса по программированию на C#, охватывающие ключевые этапы жизненного цикла программного обеспечения, принципы структурного и объектно-ориентированного подходов, базовые средства языка и приёмы проектирования.

1.1. Этапы жизненного цикла программного обеспечения

Жизненный цикл ПО — это последовательность этапов, через которые проходит программный продукт от идеи до завершения эксплуатации.

Основные этапы

  • Анализ требований. Выявление задач, потребностей пользователей, функциональных и нефункциональных требований. Пример: подготовка ТЗ для модуля «Учёт студентов» (ввод, поиск, сортировка).
  • Проектирование. Определение архитектуры, модулей, интерфейсов, классов, структур данных. Применение UML-диаграмм (классов, прецедентов, последовательностей).
  • Реализация (кодирование). Написание исходного кода на выбранном языке (C#), модульное тестирование.
  • Тестирование. Проверка корректности и полноты реализации требований (юнит, интеграционные, системные тесты).
  • Внедрение и эксплуатация. Передача пользователям, обучение, техподдержка.
  • Сопровождение и модернизация. Исправления, добавление функций, оптимизация.
Совет: для учебных проектов фиксируйте артефакты каждого этапа — ТЗ, диаграммы, README, тест-план, отчёт о внедрении. Это облегчит проверку и дальнейшую поддержку.

2. Принципы структурного программирования

Структурное программирование опирается на три базовые управляющие структуры: следование, ветвление и цикл. Код разбивается на логически завершённые функции и процедуры.

using System;

class Program
{
    static void Main()
    {
        int a = 10, b = 5;
        if (a > b)
        {
            Console.WriteLine("a больше b");
        }
        else
        {
            Console.WriteLine("a меньше или равно b");
        }

        for (int i = 1; i <= 5; i++)
        {
            Console.WriteLine($"Итерация {i}");
        }
    }
}
  • используйте управляющие структуры вместо goto;
  • соблюдайте модульность;
  • пишите читаемо и предсказуемо.

3. Основы объектно-ориентированного программирования (ООП)

ООП рассматривает программу как совокупность взаимодействующих объектов.

Инкапсуляция

class Student
{
    private string name;
    public void SetName(string n) => name = n;
    public string GetName() => name;
}

Наследование

class Person
{
    public string Name { get; set; }
}

class Student : Person
{
    public string Group { get; set; }
}

Полиморфизм

class Shape
{
    public virtual void Draw() => Console.WriteLine("Фигура");
}

class Circle : Shape
{
    public override void Draw() => Console.WriteLine("Круг");
}

class Square : Shape
{
    public override void Draw() => Console.WriteLine("Квадрат");
}

class Program
{
    static void Main()
    {
        Shape s = new Circle();
        s.Draw();  // Выведет: Круг
    }
}

+------------------+

Шаг 5. Основной модуль программы (Program.cs)

using System;
using System.Collections.Generic;
using System.Text.Json;
using System.IO;

class Program
{
    static List<Student> students = new List<Student>();
    static string filePath = "students.json";

    static void Main()
    {
        LoadData();

        bool exit = false;
        while (!exit)
        {
            Console.WriteLine("\n--- Меню управления студентами ---");
            Console.WriteLine("1. Добавить студента");
            Console.WriteLine("2. Показать всех студентов");
            Console.WriteLine("3. Найти по ID");
            Console.WriteLine("4. Редактировать данные");
            Console.WriteLine("5. Удалить студента");
            Console.WriteLine("6. Сохранить и выйти");
            Console.Write("Выбор: ");
            string choice = Console.ReadLine();

            switch (choice)
            {
                case "1": AddStudent(); break;
                case "2": ShowAll(); break;
                case "3": FindStudent(); break;
                case "4": EditStudent(); break;
                case "5": DeleteStudent(); break;
                case "6": SaveData(); exit = true; break;
                default: Console.WriteLine("Неверный выбор."); break;
            }
        }
    }

Шаг 6. Реализация функций

Добавление студента

static void AddStudent()
{
    Console.Write("Введите ID: ");
    int id = int.Parse(Console.ReadLine());
    Console.Write("Введите ФИО: ");
    string name = Console.ReadLine();
    Console.Write("Введите группу: ");
    string group = Console.ReadLine();
    Console.Write("Введите средний балл: ");
    double grade = double.Parse(Console.ReadLine());

    students.Add(new Student { Id = id, FullName = name, Group = group, AverageGrade = grade });
    Console.WriteLine("Студент добавлен.");
}

Просмотр всех

static void ShowAll()
{
    Console.WriteLine("\n--- Список студентов ---");
    foreach (var s in students)
        s.ShowInfo();
}

Поиск

static void FindStudent()
{
    Console.Write("Введите ID студента: ");
    int id = int.Parse(Console.ReadLine());
    var s = students.Find(x => x.Id == id);
    if (s != null) s.ShowInfo();
    else Console.WriteLine("Студент не найден.");
}

Редактирование

static void EditStudent()
{
    Console.Write("Введите ID для редактирования: ");
    int id = int.Parse(Console.ReadLine());
    var s = students.Find(x => x.Id == id);
    if (s != null)
    {
        Console.Write("Новое имя: ");
        s.FullName = Console.ReadLine();
        Console.Write("Новая группа: ");
        s.Group = Console.ReadLine();
        Console.Write("Новый средний балл: ");
        s.AverageGrade = double.Parse(Console.ReadLine());
        Console.WriteLine("Данные обновлены.");
    }
    else Console.WriteLine("Студент не найден.");
}

Удаление

static void DeleteStudent()
{
    Console.Write("Введите ID для удаления: ");
    int id = int.Parse(Console.ReadLine());
    students.RemoveAll(s => s.Id == id);
    Console.WriteLine("Студент удалён.");
}

Шаг 7. Работа с JSON-файлом

JSON — лёгкий текстовый формат хранения данных, используется для обмена между программами. Данные хранятся как пары «ключ–значение».

Пример JSON-файла:

[
  {
    "Id": 1,
    "FullName": "Иванов Иван",
    "Group": "ИС-23",
    "AverageGrade": 4.5
  },
  {
    "Id": 2,
    "FullName": "Петров Пётр",
    "Group": "ИС-23",
    "AverageGrade": 3.9
  }
]

Создание JSON программно

static void SaveData()
{
    string json = JsonSerializer.Serialize(students, new JsonSerializerOptions { WriteIndented = true });
    File.WriteAllText(filePath, json);
    Console.WriteLine("Данные сохранены в students.json");
}

static void LoadData()
{
    if (File.Exists(filePath))
    {
        string json = File.ReadAllText(filePath);
        students = JsonSerializer.Deserialize<List<Student>>(json);
        Console.WriteLine("Данные загружены из файла.");
    }
}

Справка:

  • Serialize — превращает коллекцию объектов в текст JSON;
  • Deserialize — восстанавливает объекты из файла;
  • WriteIndented = true — форматирует JSON для удобства чтения.

После запуска программы файл students.json появится в папке проекта (bin/Debug/net8.0/).
